## Step 2: Hardware Setup

Once your account is approved, you're ready to set up your hardware.

### 2.1 Laser Distance Meter Installation

1. **Choose mounting location:**
   - Front bumper center (recommended for clearance monitoring)
   - Vehicle roof (for overhead measurements)
   - Handheld use with tripod mount

**[IMAGE: laser-mounting-locations.jpg]**  
*Diagram showing recommended mounting positions*

2. **Secure the mounting bracket:**
   - Use provided bolts or magnetic base
   - Ensure stable, vibration-free mounting
   - Point laser forward in direction of travel

3. **Connect power cable:**
   - Connect to 12V vehicle power or battery pack
   - Verify power LED illuminates

**[IMAGE: laser-power-connection.jpg]**

4. **Connect USB-to-Serial cable:**
   - Connect serial end to LDM data port
   - Connect USB end to your tablet/laptop
   - Note the cable color/label for identification

**[IMAGE: laser-serial-connection.jpg]**

### 2.2 GPS Receiver Installation

1. **Choose mounting location:**
   - Vehicle roof (best satellite visibility)
   - Dashboard near windshield
   - Handheld mounting pole

**[IMAGE: gps-mounting-locations.jpg]**

2. **Mount the GPS receiver:**
   - Magnetic mount for metal surfaces
   - Adhesive mount for non-metal surfaces
   - Ensure clear view of sky (no obstructions)

3. **Connect power cable:**
   - Connect to 12V vehicle power or battery pack
   - Verify power LED illuminates
   - Wait 1-2 minutes for satellite lock

**[IMAGE: gps-power-led.jpg]**  
*GPS unit with power LED indicator*

4. **Connect USB-to-Serial cable:**
   - Connect serial end to GPS data port
   - Connect USB end to your tablet/laptop
   - Use different USB port than laser

**[IMAGE: gps-serial-connection.jpg]**

### 2.3 Verify Hardware Connections

**[IMAGE: complete-hardware-setup.jpg]**  
*Tablet/laptop with both devices connected*

**Checklist:**
- ✓ Laser power LED: ON
- ✓ GPS power LED: ON
- ✓ Both USB cables connected to computer
- ✓ Cables secured and won't disconnect during movement

## Step 5: Verify Hardware Connection

Once connected, verify your devices are working properly.

### 5.1 Check Laser Connection

1. Look at the **Measurement Display** panel
2. Point laser at a target (wall, object, etc.)
3. You should see:
   - ✓ **Distance reading** updating in real-time
   - ✓ **Green status indicator**
   - ✓ Reading changes when you move

**[SCREENSHOT: laser-connected-status.png]**  
*Measurement panel showing live readings*

**Troubleshooting:**
- ❌ **"No Data"** - Check cable connections, try reconnecting serial port
- ❌ **"--" dashes** - Laser may need power cycle, check power cable
- ❌ **Frozen reading** - Click "Disconnect" then "Connect" again

### 5.2 Check GPS Connection

1. Look at the **GPS Data** panel (click GPS tab if hidden)
2. Wait 1-2 minutes for satellite lock
3. You should see:
   - ✓ **Latitude/Longitude** coordinates
   - ✓ **Satellite count** (4 or more is good)
   - ✓ **Fix Quality:** "3D Fix" or better
   - ✓ Your **position marker** on the map

**[SCREENSHOT: gps-connected-status.png]**  
*GPS panel showing satellite lock*

**Troubleshooting:**
- ❌ **"No Fix"** - Move GPS to location with clear sky view
- ❌ **Low satellite count** - Wait longer, ensure GPS antenna is upright
- ❌ **"No Data"** - Check cable connections, verify power LED is on

## Step 7: Common Tasks

### 7.1 Set Ground Reference

When measuring clearances, set a ground reference height:

1. Point laser straight down at the ground
2. Click **"Set Ground Reference"** button
3. All future measurements are adjusted (Height = Laser Reading - Ground Reference)

**[SCREENSHOT: ground-reference-button.png]**

**Use Case:** When your laser is mounted 10 feet high, and you measure a 12-foot overhead wire, the display shows "2 ft clearance" (12 - 10 = 2).

## Troubleshooting Guide

### Issue: "Cannot connect to serial port"

**Solutions:**
1. ✓ Refresh the page and try again
2. ✓ Unplug USB cables, wait 5 seconds, reconnect
3. ✓ Try different USB ports on your computer
4. ✓ Check cable connections at device end
5. ✓ Restart your computer
6. ✓ Use Chrome or Edge browser (best compatibility)

### Issue: Laser shows "No Data" or dashes

**Solutions:**
1. ✓ Verify power LED is ON on laser unit
2. ✓ Check power cable connection
3. ✓ Disconnect and reconnect serial port in app
4. ✓ Power cycle the laser (unplug power, wait 10 seconds, reconnect)

### Issue: GPS shows "No Fix"

**Solutions:**
1. ✓ Wait 2-5 minutes for initial satellite lock (cold start)
2. ✓ Move GPS to location with clear sky view
3. ✓ Avoid parking garages, tunnels, dense tree cover
4. ✓ Ensure GPS antenna is upright (parallel to ground)
5. ✓ Check that GPS power LED is on

### Issue: Measurements seem incorrect

**Solutions:**
1. ✓ Verify laser is pointed at intended target
2. ✓ Check if ground reference is set correctly
3. ✓ Ensure laser lens is clean (use soft cloth)
4. ✓ Verify unit setting (metric vs imperial)
5. ✓ Check for reflective or transparent targets (may cause errors)

### Issue: App won't load or is slow

**Solutions:**
1. ✓ Check internet connection
2. ✓ Clear browser cache (Settings → Privacy → Clear Data)
3. ✓ Try different browser (Chrome recommended)
4. ✓ Close other browser tabs
5. ✓ Restart browser

### Issue: Photos not capturing

**Solutions:**
1. ✓ Grant camera permission when prompted
2. ✓ Check browser settings → Site Permissions → Camera
3. ✓ Ensure camera is not being used by another app
4. ✓ Try refreshing the page

## Safety Guidelines

### Laser Safety
- ⚠️ **Never look directly into laser beam**
- ⚠️ **Do not point at eyes or reflective surfaces**
- ⚠️ Class 2 laser - safe for momentary viewing
- Keep away from children

### Vehicle Operation
- ⚠️ **Never operate measurement system while driving**
- Always have a dedicated operator
- Ensure cables don't interfere with vehicle controls
- Secure all equipment before moving vehicle

### Electrical Safety
- Use only provided power cables and adapters
- Don't expose equipment to rain without weatherproofing
- Disconnect power when not in use for extended periods

### Connection Quick Steps
1. Connect laser USB cable → Computer
2. Connect GPS USB cable → Computer  
3. Open MeasurePRO web app
4. Click "Connect Laser" → Select port → Connect

6. Wait for GPS satellite lock (1-2 min)

### Taking Measurements
1. Create/select active survey
2. Point laser at target
3. Press **[Space]** or click "Log"
4. Add photo (optional)
5. Add note (optional)

### Keyboard Shortcuts
- **Space** — Log measurement (POI)
- **C** — Capture photo
- **V** — Start/stop video recording
- **G** — Set ground reference
- **M** — Toggle map view
- **A** — Run AI detection (MeasurePRO+ only)
- **Alt+3** — Start/stop Road Profile logging
- **Alt+E** — Toggle envelope clearance monitoring
- **Alt+L** — Connect/disconnect laser
- **Alt+G** — Connect/disconnect GPS

### Supported Laser Devices
| Device | Protocol | Baud Rate |
|---|---|---|
| SolTec-10m / 30m / 70m | SolTec binary | 115,200 8N1 |
| SolTec-2700 | SolTec binary | 115,200 8N1 |
| RSA High Pole Laser | RSA 3-byte binary | 115,200 8N1 |
| **LDM71** | **ASCII output** | **115,200 8N1** |
| Lateral / Rear lasers | Any | 19,200 7E1 |
| Mock mode | Simulated | N/A |

### GPS Priority / Failover
1. Swift Navigation Duro (RTK-GNSS) — highest accuracy
2. USB GPS dongle
3. Bluetooth GPS receiver
4. Browser geolocation (lowest accuracy)

Auto-switch occurs after 5 seconds of signal loss.
