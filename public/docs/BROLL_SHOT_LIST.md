## SHOT QUALITY REQUIREMENTS

### Technical Standards
- **Resolution:** 4K minimum (3840×2160)
- **Frame Rate:** 60fps for action, 24fps for cinematic
- **Codec:** ProRes 422 HQ or H.264 high bitrate
- **Color:** Log profile for maximum color grading flexibility
- **Audio:** 48kHz, 24-bit, dual-channel minimum

### Composition Guidelines
- **Rule of Thirds:** Apply for all static shots
- **Headroom:** Appropriate spacing for subjects
- **Leading Lines:** Use roads, cables for visual flow
- **Depth:** Utilize shallow/deep focus strategically
- **Stability:** Tripod or gimbal for all non-handheld shots

### Lighting Standards
- **Natural Light:** Golden hour preferred for outdoor beauty shots
- **Artificial Light:** 3-point lighting for interviews/product shots
- **Exposure:** Properly exposed, no clipping (highlights/shadows)
- **Consistency:** Match color temperature across scenes
