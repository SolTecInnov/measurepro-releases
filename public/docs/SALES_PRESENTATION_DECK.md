## SLIDE 6: DEMO - BASIC MEASUREMENT

### Visual:
- Live demo or screenshot of Measure tab
- Large clearance display showing real-time reading
- Color-coded status (green, yellow, red zones)

### Content:
**Real-Time Clearance Monitoring**

```
┌─────────────────────────┐
│  OVERHEAD CLEARANCE     │
│                         │
│      17.8 ft           │ ← Live reading
│                         │
│   STATUS: SAFE ✓       │ ← Green = Safe
│                         │
│  Ground Ref: 15.0 ft   │
│  Clearance: 2.8 ft     │
└─────────────────────────┘
```

**How It Works:**
1. Laser mounted on roof measures overhead distance
2. Ground reference (vehicle height) subtracted automatically
3. Clearance margin displayed in real-time
4. Color changes based on thresholds (green → yellow → red)
5. Audio alerts at warning and critical levels

### Speaker Notes:
```
Let me show you how MeasurePRO works in practice.

[DEMO IF POSSIBLE, OR WALK THROUGH SCREENSHOT]

The Measure tab is your command center. It shows your current overhead 
clearance in large, easy-to-read numbers that update two to three times 
per second. This is a live feed from the laser distance meter mounted 
on your roof.

In this example, the laser is reading 17.8 feet to the overhead obstacle. 
The vehicle is configured as 15 feet tall (the ground reference). So 
your actual clearance margin is 17.8 minus 15, which equals 2.8 feet.

The status indicator shows green because 2.8 feet is well above your 
warning threshold. If clearance dropped below 2 feet, it would turn 
yellow and beep—a "heads up" warning. If it dropped below 1 foot, it 
would turn red and sound a loud alarm—a "stop now" warning.

This real-time awareness transforms how drivers operate. Instead of 
guessing or estimating, they have precise, continuous feedback. They 
know exactly how much clearance they have at all times.

This alone prevents costly mistakes and keeps drivers safe.
```
