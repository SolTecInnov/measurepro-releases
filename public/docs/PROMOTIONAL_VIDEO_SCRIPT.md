## PRODUCTION SPECIFICATIONS

### Video Technical Requirements
- **Resolution:** 4K (3840×2160) minimum
- **Frame Rate:** 60fps for action, 24fps for cinematic
- **Aspect Ratio:** 16:9 (YouTube standard)
- **Color Space:** Rec. 709
- **Bitrate:** 50+ Mbps for master

### Audio Technical Requirements
- **Sample Rate:** 48kHz
- **Bit Depth:** 24-bit
- **Channels:** Stereo (2.0)
- **Voiceover:** Professional studio recording
- **Music:** Licensed, royalty-free or original
- **SFX:** High-quality sound effects library

### Screen Capture Requirements
- **Resolution:** 1920×1080 minimum
- **Software:** OBS Studio or similar
- **Frame Rate:** 60fps for smooth UI
- **Cursor:** Custom cursor for visibility
- **Clean:** No personal data, test accounts only

### Color Grading Notes
- Professional corporate look
- Slightly cool tone (tech feel)
- High contrast for screen visibility
- Vibrant UI colors (green alerts, red warnings)
- Consistent grade across all scenes

### Music & Sound Design
- **Scene 1-3:** Uplifting, tech-forward (120-130 BPM)
- **Scene 4-6:** Driving, energetic (130-140 BPM)
- **Scene 7-8:** Dramatic build (140+ BPM)
- **Scene 9-10:** Calm, professional (100-120 BPM)
- **Scene 11:** Inspiring finale (120 BPM)
