## TEMPLATE CUSTOMIZATION GUIDE

### Variables to Replace:
- `[First Name]` - Customer's first name
- `[Tier Name]` - MeasurePRO+, Envelope Clearance, Convoy Guardian, Route Enforcement
- `[Amount]` - Dollar amount
- `[Date]` - Formatted date
- `[Count]` - Numeric count
- `[Link]` - URL link
- `[Your App URL]` - Your Replit deployment URL

### Personalization Tips:
1. **Segmentation:** Send tier-specific content (don't mention AI features to Free users)
2. **Timing:** Adjust send times based on customer timezone
3. **Frequency:** Don't overwhelm—space emails 2-3 days apart minimum
4. **A/B Testing:** Test subject lines and CTAs
5. **Mobile Optimization:** Keep emails concise, use large buttons

### Email Metrics to Track:
- Open Rate (target: 20-30%)
- Click-Through Rate (target: 3-5%)
- Conversion Rate (for upgrade emails)
- Unsubscribe Rate (keep below 0.5%)
- Response Rate (for support/feedback emails)
