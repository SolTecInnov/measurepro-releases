New Solution: File Discovery Endpoint
MeasurePRO can call this endpoint to automatically find and register all uploaded files:

POST /api/measurepro/discover-files/:surveyId
Authorization: Bearer <api_key>

This will:

Scan Firebase Storage for all files in measurepro/{surveyId}/
Register any unregistered files in the database
Link photos/videos to POIs based on the folder structure
Update the POI's photos/videos arrays
To test it from MeasurePRO, add this after file uploads complete:

// After uploading files, call discover to register them
const discoverResponse = await this.client.post(
  `/discover-files/${roadscopeSurveyId}`
);
console.log('[RoadScope] File discovery result:', discoverResponse.data);
