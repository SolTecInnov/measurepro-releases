    Y = altitude_corrected_m
Optional:
- L_PROFILE_RAW (station vs altitude_raw_m)
- L_PROFILE_SELECTED (station vs altitude_selected_m)

This DXF is for CAD drafting, not just GIS.

E) LANDXML EXPORT (CRITICAL FOR CIVIL 3D)
Generate a LandXML file that Civil 3D can import. It must include:
- Alignment geometry (CoordGeom) in a projected coordinate system when possible
- Existing Ground Profile:

- Units (meters), and coordinate system note in metadata (LandXML itself may not fully encode EPSG)
- Name alignment and profile using project/profile names

If full alignment curve fitting is too heavy initially:
- Implement LandXML as “sampled” alignment polyline + profile points along stationing.
- Validate by importing in Civil 3D.

F) ZIP BUNDLE EXPORT (MANDATORY)
Export a ZIP that includes:
- alignment.geojson
- profile.csv

- export.landxml (optional)
