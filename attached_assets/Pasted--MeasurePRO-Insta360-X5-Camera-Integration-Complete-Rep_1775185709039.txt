# MeasurePRO — Insta360 X5 Camera Integration
## Complete Replit Agent Prompt (Final Version)

---

## CONTEXT & PHILOSOPHY

You are adding 360° camera integration to **MeasurePRO**, a road survey PWA
(React + TypeScript + Firebase) running in Chrome on a **Windows 11 tablet**
in a moving survey vehicle.

### Non-negotiable design rules

**The operator is driving.** The laser distance measurement and POI workflow
are PRIMARY. The 360° camera is a background system — but NOT an ignorable one.

**Camera failure must always be surfaced.** If the camera fails mid-survey,
the operator must be informed immediately so they can stop safely and fix the
issue — otherwise they return to the office with 0 usable footage, requiring
a costly return trip to the field.

**Error philosophy:**
- Camera WARNINGS (low battery, low storage) → non-intrusive HUD badge only
- Camera ERRORS (connection lost, recording stopped, write error) → prominent
  persistent red banner at top of screen that requires manual dismissal
- Pre-survey camera check → blocking soft-gate with live preview before
  survey starts (operator must confirm camera OK or explicitly skip)
- NEVER show modal dialogs while vehicle is in motion
- All alerts during driving are persistent banners, never popups or modals

**The ideal UX:**
1. Before survey: operator validates camera via preview + checklist
2. During survey: camera records silently, HUD shows only a small REC dot
3. If something breaks: red banner appears immediately, operator stops and fixes
4. After survey: files download automatically in the background

### Hardware setup
- Tablet: Windows 11, Chrome browser, MeasurePRO PWA
- Camera: Insta360 X5, connected via USB-C cable to the tablet
- GNSS: Swift Navigation Duro, also USB to tablet
- Internet: Starlink WiFi (always connected, used for Firebase)
- Camera control: via a Node.js Windows background service (camera-bridge)
  that sits between the PWA and the Insta360 OSC protocol over USB

---

## ARCHITECTURE

```
[Chrome — MeasurePRO PWA]
         ↕  HTTP  fetch('http://localhost:3001/...')
[camera-bridge — Node.js Windows Service]
         ↕  OSC HTTP over USB network adapter (192.168.42.1)
[Insta360 X5]    [Swift Duro GNSS]
         ↕  USB-C                ↕  USB
         └──────────[USB Hub]────┘
                        ↕
               [Windows 11 Tablet]
                        ↕  WiFi
                   [Starlink → Firebase]
```

The camera-bridge is a standalone Node.js Express server that:
- Starts automatically on Windows login (via Windows Task Scheduler)
- Runs silently in background (system tray icon only)
- Exposes REST API on localhost:3001
- Communicates with Insta360 X5 via OSC protocol over USB
  (camera enumerates as USB network adapter at 192.168.42.1)
- Is completely independent from MeasurePRO PWA
- Continues running and downloading files even if Chrome is closed

### `camera-bridge/src/cameraController.ts`

```typescript
import fetch from 'node-fetch'
import fs from 'fs'
import path from 'path'
import os from 'os'
import { logger } from './logger'

const CAMERA_URL = 'http://192.168.42.1'
const TIMEOUT_MS = 8000

interface OscCommand {
  name: string
  parameters?: Record<string, any>
}

export class CameraController {

  private async oscExecute(command: OscCommand): Promise<any> {
    const controller = new AbortController()
    const timeout = setTimeout(() => controller.abort(), TIMEOUT_MS)
    try {
      const response = await fetch(`${CAMERA_URL}/osc/commands/execute`, {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json;charset=utf-8',
          'Accept': 'application/json',
          'X-XSRF-Protected': '1'
        },
        body: JSON.stringify(command),
        signal: controller.signal
      })
      if (!response.ok) throw new Error(`OSC HTTP ${response.status}`)
      return await response.json()
    } finally {
      clearTimeout(timeout)
    }
  }

  async ping(): Promise<boolean> {
    try {
      const res = await fetch(`${CAMERA_URL}/osc/info`,
        { signal: AbortSignal.timeout(3000) })
      return res.ok
    } catch {
      return false
    }
  }

  async getInfo(): Promise<any> {
    const res = await fetch(`${CAMERA_URL}/osc/info`)
    if (!res.ok) throw new Error(`Cannot reach camera at ${CAMERA_URL}`)
    return res.json()
  }

  async getStatus(): Promise<{
    batteryPercent: number
    storageFreeMB: number
    isRecording: boolean
    recordingDuration: number
    firmwareVersion: string
  }> {
    const res = await fetch(`${CAMERA_URL}/osc/state`, {
      headers: { 'X-XSRF-Protected': '1' }
    })
    if (!res.ok) throw new Error(`OSC state error ${res.status}`)
    const data = await res.json()
    return {
      batteryPercent:    data.state?._battery ?? -1,
      storageFreeMB:     Math.round(data.state?._storageUri ?? 0),
      isRecording:       data.state?._captureStatus === 'recording',
      recordingDuration: data.state?._recordingTime ?? 0,
      firmwareVersion:   data.state?._firmwareVersion ?? 'unknown'
    }
  }

  async startRecording(options: {
    resolution?: '5.7K' | '8K' | '4K'
    fps?: 24 | 30 | 60
    mode?: 'standard' | 'timelapse_road'
  } = {}): Promise<void> {
    await this.oscExecute({
      name: 'camera.setOptions',
      parameters: {
        options: {
          captureMode: 'video',
          fileFormat: options.resolution === '8K'
            ? { type: 'video', width: 7680, height: 3840 }
            : { type: 'video', width: 5760, height: 2880 }
        }
      }
    })
    await this.oscExecute({ name: 'camera.startCapture' })
    logger.info('Recording started', options)
  }

  async stopRecording(): Promise<{ filename: string }> {
    const result = await this.oscExecute({ name: 'camera.stopCapture' })
    logger.info('Recording stopped')
    return { filename: result.results?.fileUrls?.[0] ?? '' }
  }

  async capturePhoto(): Promise<{ filename: string }> {
    // Switch to photo mode
    await this.oscExecute({
      name: 'camera.setOptions',
      parameters: { options: { captureMode: 'image' } }
    })
    const result = await this.oscExecute({ name: 'camera.takePicture' })
    // Poll until done
    let status = result
    let attempts = 0
    while (status.state === 'inProgress' && attempts < 30) {
      await new Promise(r => setTimeout(r, 500))
      const check = await fetch(`${CAMERA_URL}/osc/commands/status`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json', 'X-XSRF-Protected': '1' },
        body: JSON.stringify({ id: status.id })
      })
      status = await check.json()
      attempts++
    }
    // Switch back to video mode
    await this.oscExecute({
      name: 'camera.setOptions',
      parameters: { options: { captureMode: 'video' } }
    })
    return { filename: status.results?.fileUrls?.[0] ?? '' }
  }

  // Captures a single low-res preview image for lens inspection
  // Returns path to a local temp file
  async capturePreviewSnapshot(): Promise<{ localPath: string }> {
    await this.oscExecute({
      name: 'camera.setOptions',
      parameters: {
        options: {
          captureMode: 'image',
          fileFormat: { type: 'image', width: 3840, height: 1920 }
        }
      }
    })
    const result = await this.oscExecute({ name: 'camera.takePicture' })
    let status = result
    let attempts = 0
    while (status.state === 'inProgress' && attempts < 30) {
      await new Promise(r => setTimeout(r, 600))
      const check = await fetch(`${CAMERA_URL}/osc/commands/status`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json', 'X-XSRF-Protected': '1' },
        body: JSON.stringify({ id: status.id })
      })
      status = await check.json()
      attempts++
    }
    const fileUrl = status.results?.fileUrls?.[0]
    if (!fileUrl) throw new Error('Preview capture returned no file URL')
    // Download the preview file to a temp location
    const tempPath = path.join(os.tmpdir(), `preview_${Date.now()}.jpg`)
    const fileRes = await fetch(`${CAMERA_URL}${fileUrl}`)
    if (!fileRes.ok) throw new Error(`Failed to download preview: ${fileRes.status}`)
    const buffer = await fileRes.buffer()
    fs.writeFileSync(tempPath, buffer)
    // Switch back to video mode
    await this.oscExecute({
      name: 'camera.setOptions',
      parameters: { options: { captureMode: 'video' } }
    })
    return { localPath: tempPath }
  }

  async listFiles(): Promise<any[]> {
    const result = await this.oscExecute({
      name: 'camera.listFiles',
      parameters: { fileType: 'all', startPosition: 0, entryCount: 100 }
    })
    return result.results?.entries ?? []
  }

  async getSettings(): Promise<any> {
    return this.oscExecute({
      name: 'camera.getOptions',
      parameters: {
        optionNames: [
          'captureMode', 'fileFormat', 'iso', 'shutterSpeed',
          'whiteBalance', '_videoHDR', '_stabilization'
        ]
      }
    })
  }

  async applySettings(settings: Record<string, any>): Promise<void> {
    await this.oscExecute({
      name: 'camera.setOptions',
      parameters: { options: settings }
    })
  }
}
```

### `camera-bridge/src/fileManager.ts`

```typescript
import fs from 'fs'
import path from 'path'
import fetch from 'node-fetch'
import { CameraController } from './cameraController'
import { logger } from './logger'

const SURVEY_BASE_DIR = 'C:\\SolTec\\Surveys'

interface PhotoMeta {
  filename: string
  lat: number; lng: number; altitude: number; heading: number
  surveyId: string; poiType: string; poiLabel: string
  capturedAt: number
}

interface DownloadJob {
  jobId: string
  surveyId: string
  status: 'running' | 'done' | 'error'
  totalFiles: number
  downloadedFiles: number
  currentFile: string
  errorMessage?: string
  outputDir: string
}

export class FileManager {
  private jobs = new Map<string, DownloadJob>()
  private photoMetaBuffer: PhotoMeta[] = []

  recordPhotoMetadata(meta: PhotoMeta): void {
    this.photoMetaBuffer.push(meta)
    const dir = path.join(SURVEY_BASE_DIR, meta.surveyId)
    fs.mkdirSync(dir, { recursive: true })
    fs.writeFileSync(
      path.join(dir, 'photo_metadata.json'),
      JSON.stringify(
        this.photoMetaBuffer.filter(m => m.surveyId === meta.surveyId),
        null, 2
      )
    )
  }

  startDownload(
    surveyId: string,
    outputDir: string,
    camera: CameraController
  ): string {
    const jobId = `dl_${Date.now()}`
    const job: DownloadJob = {
      jobId, surveyId, status: 'running',
      totalFiles: 0, downloadedFiles: 0, currentFile: '',
      outputDir: outputDir || path.join(SURVEY_BASE_DIR, surveyId)
    }
    this.jobs.set(jobId, job)
    this.runDownload(job, camera).catch(e => {
      job.status = 'error'
      job.errorMessage = e.message
      logger.error('Download failed', e)
    })
    return jobId
  }

  getDownloadProgress(jobId: string): DownloadJob | { error: string } {
    return this.jobs.get(jobId) ?? { error: 'Job not found' }
  }

  private async runDownload(
    job: DownloadJob,
    camera: CameraController
  ): Promise<void> {
    fs.mkdirSync(job.outputDir, { recursive: true })
    const files = await camera.listFiles()
    job.totalFiles = files.length
    for (const file of files) {
      job.currentFile = file.name
      const destPath = path.join(job.outputDir, path.basename(file.name))
      if (!fs.existsSync(destPath)) {
        const res = await fetch(`http://192.168.42.1${file.fileUrl}`)
        const buffer = await res.buffer()
        fs.writeFileSync(destPath, buffer)
      }
      job.downloadedFiles++
      logger.info(`Downloaded ${job.downloadedFiles}/${job.totalFiles}: ${file.name}`)
    }
    fs.writeFileSync(
      path.join(job.outputDir, 'survey_manifest.json'),
      JSON.stringify({
        surveyId: job.surveyId,
        downloadedAt: Date.now(),
        files: files.map((f: any) => f.name)
      }, null, 2)
    )
    job.status = 'done'
    logger.info(`Download complete for survey ${job.surveyId}`)
  }
}
```

### `src/services/cameraService.ts`

```typescript
const BRIDGE = 'http://localhost:3001'
const FETCH_TIMEOUT = 6000

async function bridgeFetch(path: string, options?: RequestInit): Promise<any> {
  const controller = new AbortController()
  const timeout = setTimeout(() => controller.abort(), FETCH_TIMEOUT)
  try {
    const res = await fetch(`${BRIDGE}${path}`, {
      ...options,
      signal: controller.signal,
      headers: { 'Content-Type': 'application/json', ...options?.headers }
    })
    return await res.json()
  } catch (e: any) {
    if (e.name === 'AbortError') throw new Error('Camera bridge timeout')
    if (e.message.includes('fetch')) throw new Error('Camera bridge offline')
    throw e
  } finally {
    clearTimeout(timeout)
  }
}

export const cameraService = {
  async checkBridge(): Promise<boolean> {
    try {
      const res = await bridgeFetch('/health')
      return res.status === 'ok'
    } catch { return false }
  },

  async checkCamera(): Promise<boolean> {
    try {
      const res = await bridgeFetch('/camera/ping')
      return res.connected === true
    } catch { return false }
  },

  async getStatus() {
    return bridgeFetch('/camera/status')
  },

  async startRecording(options: CameraRecordingOptions = {}) {
    return bridgeFetch('/camera/start-recording', {
      method: 'POST',
      body: JSON.stringify(options)
    })
  },

  async stopRecording() {
    return bridgeFetch('/camera/stop-recording', { method: 'POST' })
  },

  async capturePhotoPOI(geoData: CameraPhotoPOIData) {
    return bridgeFetch('/camera/capture-photo', {
      method: 'POST',
      body: JSON.stringify(geoData)
    })
  },

  // Returns the preview image as a blob URL for display
  async getPreviewSnapshotUrl(): Promise<string> {
    const res = await fetch(`${BRIDGE}/camera/preview-snapshot`, {
      signal: AbortSignal.timeout(20000) // preview can take up to 15s
    })
    if (!res.ok) throw new Error(`Preview failed: ${res.status}`)
    const blob = await res.blob()
    return URL.createObjectURL(blob)
  },

  async startPostSurveyDownload(surveyId: string) {
    return bridgeFetch('/camera/download-survey', {
      method: 'POST',
      body: JSON.stringify({
        surveyId,
        outputDir: `C:\\SolTec\\Surveys\\${surveyId}`
      })
    })
  },

  async getDownloadProgress(jobId: string) {
    return bridgeFetch(`/camera/download-status/${jobId}`)
  },

  async getSettings() {
    return bridgeFetch('/camera/settings')
  },

  async saveSettings(settings: CameraSettings) {
    return bridgeFetch('/camera/settings', {
      method: 'POST',
      body: JSON.stringify(settings)
    })
  }
}
```

```typescript
import {

const DEFAULT_SETTINGS: CameraSettings = {

export function useCameraControl() {
  const [bridgeOnline, setBridgeOnline]           = useState(false)
  const [cameraConnected, setCameraConnected]     = useState(false)
  const [status, setStatus]                       = useState<CameraStatus | null>(null)
  const [settings, setSettings]                   = useState<CameraSettings>(DEFAULT_SETTINGS)
  const [cameraError, setCameraError]             = useState<CameraError | null>(null)
  const [cameraWarnings, setCameraWarnings]       = useState<string[]>([])
  const [isCapturingPhoto, setIsCapturingPhoto]   = useState(false)
  const [lastPhotoResult, setLastPhotoResult]     = useState<string | null>(null)
  const [downloadJobId, setDownloadJobId]         = useState<string | null>(null)
  const [downloadProgress, setDownloadProgress]   = useState<any>(null)
  const wasRecordingRef                           = useRef(false)
  const activeSurveyIdRef                        = useRef<string | null>(null)
  const pollRef                                   = useRef<ReturnType<typeof setInterval> | null>(null)

  // Load settings from localStorage on mount
  useEffect(() => {
    const saved = localStorage.getItem('measurepro_camera_settings')
    if (saved) {
      try { setSettings(JSON.parse(saved)) } catch {}
    }
  }, [])

  // Poll every 10 seconds
  useEffect(() => {
    checkConnections()
    pollRef.current = setInterval(checkConnections, 10000)
    return () => { if (pollRef.current) clearInterval(pollRef.current) }
  }, [])

  // Poll download progress
  useEffect(() => {
    if (!downloadJobId) return
    const interval = setInterval(async () => {
      const progress = await cameraService.getDownloadProgress(downloadJobId)
      setDownloadProgress(progress)
      if (progress.status === 'done' || progress.status === 'error') {
        clearInterval(interval)
      }
    }, 2000)
    return () => clearInterval(interval)
  }, [downloadJobId])

  async function checkConnections() {
    const bridge = await cameraService.checkBridge()
    setBridgeOnline(bridge)

    if (!bridge) {
      // If bridge went offline while survey was active, that's an error
      if (activeSurveyIdRef.current && cameraConnected) {
        setCameraError({
          code: 'CONNECTION_LOST',
          message: 'Service caméra déconnecté!',
          detail: 'Le service camera-bridge ne répond plus.',
          recoverable: true,
          action: 'Arrêtez le véhicule. Vérifiez le câble USB-C de la caméra.'
        })
      }
      setCameraConnected(false)
      setStatus(null)
      return
    }

    const cam = await cameraService.checkCamera()
    setCameraConnected(cam)

    if (!cam) {
      if (activeSurveyIdRef.current) {
        setCameraError({
          code: 'CONNECTION_LOST',
          message: 'Caméra 360° déconnectée!',
          detail: 'La caméra X5 ne répond plus sur USB.',
          recoverable: true,
          action: 'Arrêtez le véhicule. Vérifiez le câble USB-C.'
        })
      }
      setStatus(null)
      return
    }

    try {
      const s = await cameraService.getStatus()
      if (!s.success) return

      const newStatus: CameraStatus = s.data

      // Detect recording stopped unexpectedly mid-survey
      if (
        activeSurveyIdRef.current &&
        wasRecordingRef.current &&
        !newStatus.isRecording
      ) {
        setCameraError({
          code: 'RECORDING_STOPPED',
          message: 'Enregistrement 360° interrompu!',
          detail: 'La caméra a arrêté d\'enregistrer pendant le survey.',
          recoverable: true,
          action: 'Arrêtez le véhicule. Vérifiez le stockage et le câble USB.'
        })
      }

      wasRecordingRef.current = newStatus.isRecording

      // Non-blocking warnings (HUD only, no banner)
      const warnings: string[] = []
      if (newStatus.batteryPercent < 20 && newStatus.batteryPercent >= 0) {
        warnings.push(`Batterie faible: ${newStatus.batteryPercent}%`)
      }
      if (newStatus.storageFreeMB < 1024) {
        warnings.push(`Stockage faible: ${(newStatus.storageFreeMB / 1024).toFixed(1)}GB`)
      }
      // Storage full is an error, not just a warning
      if (newStatus.storageFreeMB < 100 && activeSurveyIdRef.current) {
        setCameraError({
          code: 'STORAGE_FULL',
          message: 'Stockage caméra presque plein!',
          detail: `Seulement ${newStatus.storageFreeMB}MB restants.`,
          recoverable: false,
          action: 'Impossible de continuer. Libérez de l\'espace sur la carte SD.'
        })
      }

      setCameraWarnings(warnings)
      setStatus(newStatus)
    } catch (e) {
      // Status fetch failed — not critical unless mid-survey
    }
  }

  // Called at survey start — sets active survey context for error detection
  const startForSurvey = useCallback(async (surveyId: string) => {
    activeSurveyIdRef.current = surveyId
    if (!settings.autoStartWithSurvey || !cameraConnected) return

    try {
      const result = await cameraService.startRecording({
        resolution: settings.resolution,
        fps: settings.fps
      })
      if (!result.success) throw new Error(result.error || 'Unknown error')
      setCameraError(null)
      wasRecordingRef.current = true
    } catch (e: any) {
      setCameraError({
        code: 'START_FAILED',
        message: 'La caméra 360° n\'a pas démarré.',
        detail: e.message,
        recoverable: true,
        action: 'Vérifiez le câble USB-C et cliquez Réessayer.'
      })
    }
  }, [settings, cameraConnected])

  // Called at survey stop
  const stopForSurvey = useCallback(async (surveyId: string) => {
    activeSurveyIdRef.current = null
    wasRecordingRef.current = false
    setCameraError(null)

    if (!cameraConnected) return

    if (settings.autoStopWithSurvey && status?.isRecording) {
      try {
        await cameraService.stopRecording()
      } catch (e) {
        // Non-critical at this point — survey is already done
        console.warn('Camera stop failed:', e)
      }
    }

    if (settings.autoDownloadAfterSurvey) {
      try {
        const result = await cameraService.startPostSurveyDownload(surveyId)
        if (result.jobId) setDownloadJobId(result.jobId)
      } catch (e) {
        console.warn('Auto-download failed:', e)
      }
    }
  }, [settings, cameraConnected, status])

  // Called when operator taps the manual photo POI button
  const capturePhotoPOI = useCallback(async (geoData: {
    lat: number; lng: number; altitude: number; heading: number
    surveyId: string; poiType: string; poiLabel: string
  }) => {
    if (!cameraConnected || isCapturingPhoto) return null
    setIsCapturingPhoto(true)
    try {
      const result = await cameraService.capturePhotoPOI(geoData)
      if (result.success) {
        setLastPhotoResult(result.filename)
        setTimeout(() => setLastPhotoResult(null), 3000)
        return result.filename
      }
      throw new Error(result.error || 'Photo failed')
    } catch (e: any) {
      setCameraError({
        code: 'WRITE_ERROR',
        message: 'Échec de la photo 360°.',
        detail: e.message,
        recoverable: true,
        action: 'Réessayez. Vérifiez le stockage de la caméra.'
      })
      return null
    } finally {
      setIsCapturingPhoto(false)
    }
  }, [cameraConnected, isCapturingPhoto])

  // Retry after an error
  const retryAfterError = useCallback(async () => {
    if (!cameraError) return
    setCameraError(null)

    if (cameraError.code === 'START_FAILED' || cameraError.code === 'RECORDING_STOPPED') {
      if (activeSurveyIdRef.current) {
        await startForSurvey(activeSurveyIdRef.current)
      }
    } else {
      await checkConnections()
    }
  }, [cameraError, startForSurvey])

  const dismissError = useCallback(() => {
    setCameraError(null)
  }, [])

  const saveSettings = useCallback(async (newSettings: CameraSettings) => {
    setSettings(newSettings)
    localStorage.setItem('measurepro_camera_settings', JSON.stringify(newSettings))
    try { await cameraService.saveSettings(newSettings) } catch {}
  }, [])

  return {
    // Connection state
    bridgeOnline,
    cameraConnected,
    status,
    settings,
    // Errors and warnings
    cameraError,
    cameraWarnings,
    // Photo capture
    isCapturingPhoto,
    lastPhotoResult,
    // Post-survey download
    downloadProgress,
    // Actions
    startForSurvey,
    stopForSurvey,
    capturePhotoPOI,
    retryAfterError,
    dismissError,
    saveSettings,
    refreshStatus: checkConnections
  }
}
```

### `src/components/CameraHUD.tsx`

Minimal non-intrusive HUD. Shows only what is needed.
Warnings (battery, storage) appear in the badge.
Errors are handled by CameraErrorBanner, not here.

```tsx
import { useCameraControl } from '../hooks/useCameraControl'

interface CameraHUDProps {
  currentPosition: { lat: number; lng: number; altitude: number; heading: number }
  currentSurveyId: string | null
  currentPOIType: string
  currentPOILabel: string
}

export function CameraHUD({
  currentPosition,
  currentSurveyId,
  currentPOIType,
  currentPOILabel
}: CameraHUDProps) {
  const {
    bridgeOnline, cameraConnected, status, settings,
    cameraWarnings, isCapturingPhoto, lastPhotoResult,
    capturePhotoPOI
  } = useCameraControl()

  if (!settings.showHUD) return null
  if (!bridgeOnline) return null

  const positionStyle: React.CSSProperties = {
    position: 'fixed',
    zIndex: 900,
    ...(settings.hudPosition === 'top-right'    && { top: 80, right: 16 }),
    ...(settings.hudPosition === 'top-left'     && { top: 80, left: 16 }),
    ...(settings.hudPosition === 'bottom-right' && { bottom: 80, right: 16 }),
    ...(settings.hudPosition === 'bottom-left'  && { bottom: 80, left: 16 }),
  }

  // Camera not connected (but bridge is running) — show warning badge
  if (!cameraConnected) {
    return (
      <div style={positionStyle}>
        <div className="camera-hud-badge camera-hud-badge--warning">
          <span>📷 Caméra non détectée</span>
        </div>
      </div>
    )
  }

  return (
    <div style={positionStyle} className="camera-hud">

      {/* Status badge */}
      <div className={`camera-hud-badge ${cameraWarnings.length > 0 ? 'camera-hud-badge--warn' : 'camera-hud-badge--ok'}`}>
        {status?.isRecording && (
          <span className="camera-hud-rec-dot" aria-label="Enregistrement actif" />
        )}
        {cameraWarnings.length > 0 ? (
          // Show first warning in badge
          <span className="camera-hud-warn-text">{cameraWarnings[0]}</span>
        ) : (
          status?.isRecording && (
            <span className="camera-hud-duration">
              {formatDuration(status.recordingDuration)}
            </span>
          )
        )}
      </div>

      {/* Photo POI button — only during active survey */}
      {currentSurveyId && (
        <button
          className={[
            'camera-hud-photo-btn',
            isCapturingPhoto ? 'camera-hud-photo-btn--capturing' : '',
            lastPhotoResult  ? 'camera-hud-photo-btn--success'   : ''
          ].join(' ')}
          onClick={() => capturePhotoPOI({
            ...currentPosition,
            surveyId: currentSurveyId,
            poiType: currentPOIType,
            poiLabel: currentPOILabel
          })}
          disabled={isCapturingPhoto}
          aria-label="Capturer photo 360° au POI"
          title="Photo 360° à cette position"
        >
          {isCapturingPhoto ? '⏳' : lastPhotoResult ? '✓' : '📷'}
        </button>
      )}

    </div>
  )
}

function formatDuration(seconds: number): string {
  const h = Math.floor(seconds / 3600)
  const m = Math.floor((seconds % 3600) / 60)
  const s = seconds % 60
  if (h > 0) return `${h}:${String(m).padStart(2,'0')}:${String(s).padStart(2,'0')}`
  return `${m}:${String(s).padStart(2,'0')}`
}
```

### `src/components/CameraPreCheckModal.tsx`

Shown before survey starts when camera is connected.
Operator validates lenses via live preview + checklist.
Cannot be skipped without explicit "Démarrer sans caméra" click.

```tsx
import { useState, useEffect } from 'react'
import { cameraService } from '../services/cameraService'
import { useCameraControl } from '../hooks/useCameraControl'

interface CameraPreCheckModalProps {
  isOpen: boolean
  onConfirmWithCamera: () => void
  onSkipCamera: () => void
}

export function CameraPreCheckModal({
  isOpen,
  onConfirmWithCamera,
  onSkipCamera
}: CameraPreCheckModalProps) {
  const { status, cameraConnected } = useCameraControl()

  const [previewUrl,     setPreviewUrl]     = useState<string | null>(null)
  const [previewLoading, setPreviewLoading] = useState(false)
  const [previewError,   setPreviewError]   = useState<string | null>(null)

  const [checklist, setChecklist] = useState({
    lensClean:    false,
    mountSecure:  false,
    batteryOk:    false,
    storageOk:    false
  })

  // Auto-fill battery and storage from live status
  useEffect(() => {
    if (status) {
      setChecklist(c => ({
        ...c,
        batteryOk: status.batteryPercent > 20,
        storageOk: status.storageFreeMB  > 2048
      }))
    }
  }, [status])

  // Cleanup preview blob URL when modal closes
  useEffect(() => {
    if (!isOpen && previewUrl) {
      URL.revokeObjectURL(previewUrl)
      setPreviewUrl(null)
    }
  }, [isOpen])

  async function loadPreview() {
    setPreviewLoading(true)
    setPreviewError(null)
    try {
      const url = await cameraService.getPreviewSnapshotUrl()
      setPreviewUrl(url)
    } catch (e: any) {
      setPreviewError('Impossible de charger le preview: ' + e.message)
    } finally {
      setPreviewLoading(false)
    }
  }

  function toggle(key: keyof typeof checklist) {
    setChecklist(c => ({ ...c, [key]: !c[key] }))
  }

  const allChecked = Object.values(checklist).every(Boolean)

  if (!isOpen) return null

  return (
    <div className="camera-precheck-overlay" role="dialog" aria-modal="true">
      <div className="camera-precheck-modal">

        <h2 className="camera-precheck-title">
          Vérification caméra 360° avant survey
        </h2>

        {/* Live preview section */}
        <div className="camera-precheck-preview">
          {!previewUrl && !previewLoading && !previewError && (
            <button
              className="preview-load-btn"
              onClick={loadPreview}
              disabled={!cameraConnected}
            >
              📷 Charger le preview (vérification des lentilles)
            </button>
          )}

          {previewLoading && (
            <div className="preview-loading">
              <span className="preview-loading-spinner" />
              Capture en cours... (~10-15 secondes)
            </div>
          )}

          {previewError && (
            <div className="preview-error">
              <span>{previewError}</span>
              <button onClick={loadPreview} className="preview-retry-btn">
                Réessayer
              </button>
            </div>
          )}

          {previewUrl && (
            <div className="preview-image-wrap">
              {/* Show equirectangular flat — sufficient for lens check */}
              <img
                src={previewUrl}
                alt="Preview caméra 360° — vérification des lentilles"
                className="camera-precheck-image"
              />
              <div className="preview-image-footer">
                <p>Vérifiez que les deux lentilles sont propres sur toute leur surface.</p>
                <button onClick={loadPreview} className="preview-refresh-btn">
                  ↺ Actualiser le preview
                </button>
              </div>
            </div>
          )}
        </div>

        {/* Checklist */}
        <div className="camera-precheck-list">
          <h3>Checklist pré-départ</h3>

          <label className="precheck-item">
            <input
              type="checkbox"
              checked={checklist.lensClean}
              onChange={() => toggle('lensClean')}
            />
            <span>Lentilles propres et sans obstruction (eau, boue, doigt)</span>
          </label>

          <label className="precheck-item">
            <input
              type="checkbox"
              checked={checklist.mountSecure}
              onChange={() => toggle('mountSecure')}
            />
            <span>Mount de toit solidement fixé, câble USB-C bien branché</span>
          </label>

          <label className={`precheck-item ${!checklist.batteryOk ? 'precheck-item--auto-warn' : 'precheck-item--auto-ok'}`}>
            <input
              type="checkbox"
              checked={checklist.batteryOk}
              readOnly
              disabled
            />
            <span>
              {checklist.batteryOk
                ? `Batterie suffisante (${status?.batteryPercent ?? '?'}%)`
                : `Batterie faible (${status?.batteryPercent ?? '?'}%) — rechargez avant de partir`
              }
            </span>
          </label>

          <label className={`precheck-item ${!checklist.storageOk ? 'precheck-item--auto-warn' : 'precheck-item--auto-ok'}`}>
            <input
              type="checkbox"
              checked={checklist.storageOk}
              readOnly
              disabled
            />
            <span>
              {checklist.storageOk
                ? `Stockage suffisant (${((status?.storageFreeMB ?? 0) / 1024).toFixed(1)} GB libres)`
                : `Stockage insuffisant (${((status?.storageFreeMB ?? 0) / 1024).toFixed(1)} GB) — libérez de l'espace`
              }
            </span>
          </label>
        </div>

        {/* Action buttons */}
        <div className="camera-precheck-actions">
          <button
            className="precheck-btn precheck-btn--confirm"
            onClick={onConfirmWithCamera}
            disabled={!allChecked}
          >
            ✓ Caméra prête — Démarrer le survey
          </button>
          <button
            className="precheck-btn precheck-btn--skip"
            onClick={onSkipCamera}
          >
            Démarrer sans caméra 360°
          </button>
        </div>

      </div>
    </div>
  )
}
```

### `src/components/CameraDownloadBanner.tsx`

```tsx
import { useCameraControl } from '../hooks/useCameraControl'

export function CameraDownloadBanner() {
  const { downloadProgress } = useCameraControl()

  if (!downloadProgress) return null
  if (downloadProgress.status === 'done') return null

  const percent = downloadProgress.totalFiles > 0
    ? Math.round((downloadProgress.downloadedFiles / downloadProgress.totalFiles) * 100)
    : 0

  const isError = downloadProgress.status === 'error'

  return (
    <div className={`camera-download-banner ${isError ? 'camera-download-banner--error' : ''}`}>
      <span className="camera-download-banner__icon">📷</span>
      <div className="camera-download-banner__content">
        <span className="camera-download-banner__label">
          {isError
            ? `Erreur téléchargement vidéo: ${downloadProgress.errorMessage}`
            : `Téléchargement vidéo 360° en cours — ${percent}% (${downloadProgress.downloadedFiles}/${downloadProgress.totalFiles} fichiers)`
          }
        </span>
        {!isError && (
          <div className="camera-download-banner__bar">
            <div
              className="camera-download-banner__fill"
              style={{ width: `${percent}%` }}
            />
          </div>
        )}
      </div>
    </div>
  )
}
```
