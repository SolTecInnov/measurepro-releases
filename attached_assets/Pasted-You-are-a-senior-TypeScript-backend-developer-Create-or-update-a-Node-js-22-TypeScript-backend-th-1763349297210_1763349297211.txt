GNSS SAMPLE TYPE:
export interface GnssSample {
timestamp: string;
lat: number;
lon: number;
alt_m: number | null;
speed_mps: number | null;
heading_deg: number | null;
quality: number | null;
hdop: number | null;
num_sats: number | null;
source: ‘duro’ | ‘usb’ | ‘browser’;
sessionId?: string;
}

FIRESTORE DESIGN:

Collection: gnssSamples
Each document:
	•	All fields of GnssSample
	•	created_at (server timestamp)

Collection: roadProfiles
Document fields:
	•	start, end
	•	step_m
	•	grade_trigger_pct
	•	summary (total distance, climb, descent, max up/down)
	•	points (or compressed representation)

Subcollection: roadProfiles/{profileId}/sections
Each section contains:
	•	fromDistance_m
	•	toDistance_m
	•	summary
	•	optional label

Collection: gradeEvents
Each event contains:
	•	sessionId
	•	profileId (optional)
	•	direction (‘up’ or ‘down’)
	•	trigger_pct
	•	max_grade_pct
	•	start_distance_m
	•	end_distance_m
	•	start_timestamp
	•	end_timestamp
	•	created_at
