Hardware Layer

Vertical Clearance Laser

  • RSA laser — 3-byte binary protocol via USB serial
  • Jenoptik LDM71 — ASCII protocol via USB serial
  • Connects via Web Serial API (Chrome/Edge required)
  • Protocol auto-detected or manually selected in Settings

GNSS / GPS Sources

  • Swift Navigation Duro — Ethernet TCP bridge → RTK GNSS + full IMU (roll, pitch, heading)
  • USB GPS dongle — NMEA via Web Serial
  • Bluetooth GPS — NMEA via Web Bluetooth
  • Browser Geolocation — fallback only, no logging triggers

Multi-Laser (Lateral / Rear)

  • Up to 4 independent laser ports: vertical, left lateral, right lateral, rear
  • Protocol: soltec-old (19200 baud, 7E1)
  • Lateral width: single or dual laser mode
  • Rear overhang: wind blade transport monitoring up to 80 m

3D LiDAR (Hesai Pandar40P)

  • UDP packets received by Windows companion service (lidar-service/)
  • Service bridges to PWA via WebSocket on port 17777
  • Streams metrics at 10 Hz: road width, clearance at heights, alerts
  • Exports LAZ / LAS point cloud files

GPS Priority Architecture

PrioritySourceAccuracyCondition
1 — HighestSwift Duro (RTK GNSS)±2 cm (RTK fix)Active when Duro sends data within last 5 seconds
2 — MediumUSB GPS Dongle±1–3 mActive when Duro not connected or silent >5s
3 — LowBluetooth GPS±1–5 mActive when USB GPS unavailable

Priority switching is automatic with no operator action required. If the Duro loses signal mid-survey, USB GPS takes over within 5 seconds.

Data Storage — Dual Database

The system uses two database namespaces to allow safe migration from legacy data:

  • measurepro — legacy database (older surveys, migrated on read)
  • measurepro-v2 — current database (all new surveys, POIs, road profile samples)

A migration adapter handles transparent reads across both. New writes always go to -v2.

Specialist Modules (MeasurePRO+)

Envelope Clearance

ZED 2i stereo camera via WebSocket server. Real-time object detection with TensorFlow.js COCO-SSD. Automatic clearance alerts.

Convoy Guardian

Multi-vehicle coordination over WebSocket. Oversized convoy operations with shared route awareness.

Permitted Route Enforcement

GPS-based route compliance. Alerts when the vehicle deviates from the permitted path.

Swept Path Analysis

Real-time road boundary detection + multi-segment vehicle modelling. Checks whether a vehicle's turning path fits within the road envelope.
